# Expense_Tracker

A Flutter Expense Tracker application designed to help users manage and visualize their daily spending effectively.

## Features

- Add, edit, and delete expenses
- View a list of recent transactions
- Display spending charts for better insights
- Filter expenses by date
- Store data locally for offline use
- Simple and modern Material Design interface

## Technologies Used

- Flutter  
- Dart  
- Provider or setState for state management  
- Material Design Components  


### Prerequisites

Make sure you have the following installed:
- Flutter SDK  
- Android Studio or VS Code  
- A connected device or emulator
